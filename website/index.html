Get Started in 30 Seconds

Step 1 — Install
# Requires Go 1.26+
go install github.com/creydr/ai-mux/cmd/ai-mux@latest
Step 2 — Configure
# ~/.config/ai-mux/config.yaml
repos:
  - name: owner/my-repo
    path: ~/development/my-repo

agents:
  - name: claude
    command: claude
Step 3 — Launch
ai-mux daemon start --background
